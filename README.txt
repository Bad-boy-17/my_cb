 My AI Chatbot Project

🚀 Live Demo
https://my-cb.onrender.com/

 📝 Description

This is an AI-powered chatbot built as a full-stack project. The application demonstrates an end-to-end development process from front-end design to back-end logic and deployment. The chatbot can engage in dynamic, contextual conversations by leveraging a modern AI API.

 ✨ Key Features

* **AI-Powered Responses:** Utilizes the **Google Gemini API** to generate intelligent and contextually relevant responses.
* **Conversational Memory:** The chatbot can remember past messages in the conversation, allowing for a more natural and fluid dialogue.
* **Professional UI/UX:** A modern, visually appealing front-end with a **Glassmorphism** design, custom fonts, and subtle animations.
* **API Security:** The project follows best practices by securing the API key using environment variables.
* **Deployment:** The application is deployed on **Render** and is accessible 24/7.

 🛠️ Tech Stack

* **Back-end:** Python, Flask
* **Front-end:** HTML, CSS (Tailwind CSS), JavaScript
* **Database:** LocalStorage (for chat history persistence)
* **AI:** Google Gemini API
* **Deployment:** Render
* **Version Control:** Git & GitHub

## 🏃 Getting Started

To run this project locally, follow these steps:

1.  Clone the repository: `git clone https://github.com/Bad-boy-17/my_cb
2.  Navigate to the project directory: `cd my-chatbot`
3.  Create and activate a virtual environment: `python -m venv venv` and `source venv/bin/activate`
4.  Install the required packages: `pip install -r requirements.txt`
5.  Create a `.env` file and add your Google API key: `GOOGLE_API_KEY=your_key_here`
6.  Run the Flask application: `flask run`
